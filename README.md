# 🛒 YExpress E-Commerce Web Application

A modern **online shopping platform** that allows customers to browse products, add them to a cart, place orders, and track their purchases — while providing administrators with a powerful dashboard to manage products, users, and orders.

This project is designed to deliver a **real-world e-commerce experience**, similar to platforms like Amazon or Jumia.

---

## 📌 Project Overview

This system consists of two main parts:

- **Customer-facing application** – for browsing, shopping, and ordering products
- **Admin dashboard** – for managing products, orders, users, and categories

The goal is to provide a **simple, fast, and user-friendly shopping experience** while giving store owners full control over their business.

---

## ✨ Key Features

### 👤 Customer Features
- Browse products by category
- View detailed product information and images
- Select product variations (size, color, storage, etc.)
- Add products to cart
- Update cart quantities or remove items
- Checkout and place orders
- Choose payment methods (Card, Chapa, Cash on Delivery)
- Track order status (Pending → Processing → Shipped → Delivered)
- Manage user profile and view order history

---

### 🛠 Admin Features
- Add, update, and delete products
- Upload product images
- Manage product categories and variants
- View and manage customer orders
- Update order status
- Manage users
- Soft-delete products and orders for safety
